The Math

The Combined Model Equation

Three data sources combined into one framework. Actual goals capture finishing ability, xG captures chance quality, and PPDA captures pressing intensity.

// Step 1: Blend actual goals with expected goals
BlendedGF = 0.69 × ActualGoals + (-0.04) × xG
BlendedGA = 0.63 × GoalsAgainst + 0.08 × xGA

// Step 2: Adjust for pressing intensity
PressAdj = 1 + 0.155 × (LeagueAvgPPDA − TeamPPDA) / LeagueAvgPPDA

// Step 3: Pythagorean prediction with pressing adjustment
PredPTS = 2.98 × (BlendedGF^1.33 / (BlendedGF^1.33 + BlendedGA^1.35)) × 38 × PressAdj

Limitations

8 parameters on 20 data points is below the 10:1 ratio. Cross-season validation mitigates but does not eliminate overfitting risk.
Poisson model predicted 0/84 draws. Dixon-Coles with a correlation parameter would fix this known weakness.
PPDA is a proxy for pressing. GPS tracking data (distance, sprints) would be more precise but is not freely available.
On 2023/24, simple Pythagorean (2.80) narrowly beat the combined model (2.97). Better on average, not every season.
